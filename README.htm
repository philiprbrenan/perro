Files

Autocad Files

The folder obj contains the three dimensional mesh files. It should be possible to open these files using autoCad as they can be opened by freeCad.

Svg Files

The folder svg contains two dimensional representations of the triangulations of the the slices. They can be displayed in any web browser.

Image Files

The folders images and images3 contains the input images. The 3 version is a larger version of each image to make it easier to trace the outline of each shape.

coordinates2.pl

Extract the coordinates from an image.

createSlices.pl

Convert three dimesnional meshes for each slice.
